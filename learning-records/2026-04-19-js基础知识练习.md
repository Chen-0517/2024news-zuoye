## 学习内容

基于 mastery 掌握学习法，通过对话形式学习了以下7个知识点：

### 1. 变量声明：var vs let vs const

- **var**：函数作用域，可以重复声明（不推荐）
- **let**：块级作用域，不能重复声明
- **const**：块级作用域，不能重新赋值（默认选择）

### 2. JavaScript基本数据类型

| 类型 | 例子 |
|------|------|
| string | `"Hello"` |
| number | `30`, `3.14` |
| boolean | `true`, `false` |
| null | `null` |
| undefined | `undefined` |

### 3. 特殊数据类型

- **BigInt**：处理超大数字，数字后加 `n`
- **Symbol**：创建唯一的值

### 4. 算术运算符

| 运算符 | 意思 |
|--------|------|
| + | 加 |
| - | 减 |
| * | 乘 |
| / | 除 |
| % | 取余 |

### 5. 比较运算符与逻辑运算符

- **比较运算符**：`>` `<` `>=` `<=` `===` `!==`
- **逻辑运算符**：`&&` 并且、`||` 或者、`!` 取反

> 💡 建议使用 `===` 而不是 `==`（更安全）

### 6. 流程控制：if/else

```javascript
if (条件) {
  // 条件为true执行
} else {
  // 条件为false执行
}
```

### 7. 函数定义与调用

```javascript
function greet(name) {
  return "Hello, " + name + "!";
}

console.log(greet("Alice")); // Hello, Alice!
```

---

## 学习要点

1. **const 是默认选择**，不想改的变量用它
2. **let 是块级作用域**，`{}` 里的是独立的
3. **BigInt 和 Number 是两种不同类型**，不能用 `===` 比较
4. **用 `===` 严格比较**，避免自动类型转换的坑
5. **函数可以重复调用**，传入不同参数得到不同结果
